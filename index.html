    Dijkstra Algorithm

    		 Dijkstra's Algorithm is a pathfinding algorithm that generates every single route through the graph, and then selects the route that has the lowest overall cost. This works by iteratively calculating a distance for each node in the graph, starting from the start node, and continuing until we reach the end node.
